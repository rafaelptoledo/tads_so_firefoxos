Informações técnicas (Utilização)

Por ser baseado em padrões e formatos abertos, em especial, da Web, os principais elementos suportados pelo Firefox OS envolvem as mesmas características.

Gráficos:

Para quem quer desenvolver jogos e necessita de suporte gráfico, o Firefox OS suporta 2d Canvas e WebGL.

Proteção de Código:

Como toda a plataforma de aplicativos é baseada em HTML, CSS e Javascript, o código fonte não será compilado, mas sim aberto. A Mozilla recomenda que o código javascript utilize técnica de embaralhamento de código, conhecido em inglês como obfuscation, o tornando mais difícil de ser roubado. Onde for possível, a empresa também recomenda a utilização de um sistema de identificação como o Mozilla Persona, que requisita autenticação antes de fornecer conteúdo importante do backend.
